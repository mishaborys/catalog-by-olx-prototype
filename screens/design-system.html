Form Inputs

Bootstrap 5 base classes. No placeholders — labels always visible above the field.

Text / Email
Enter a valid email address.

Error state: form-control is-invalid
Label: form-label fw-500 mb-1

border: 1px solid #dee2e6
padding: .375rem .75rem

         box-shadow: 0 0 0 .25rem rgba(13,110,253,.25)
Password + Toggle + Strength Bar
Wrapper: input-group
Toggle btn: btn btn-outline-secondary

Strength bar: .pw-bar
  display: flex; gap: 4px; margin-top: .35rem
Segment: .pw-seg
  flex: 1; height: 4px; border-radius: 2px
  default bg: #e0e0e0
  active: set bg inline (red/orange/yellow/green)
Select — Standard
Bootstrap: form-select
Label: form-label fw-500 mb-1
Select — With Search
Wrapper: dropdown w-100
Trigger: form-select (styled as input)
  data-bs-toggle="dropdown"
  data-bs-auto-close="outside"
Menu: dropdown-menu p-2 w-100
Search: form-control form-control-sm
Options: dropdown-item

JS: filter options by textContent
    on input oninput event
Select — Multiselect with Checkboxes
Trigger: form-select
  data-bs-auto-close="outside"
  (keeps menu open on checkbox click)

Items: form-check inside dropdown-menu

JS: collect checked values,
    join with ', ' → display in trigger
Checkbox
Bootstrap: form-check
  form-check-input (type="checkbox")
  form-check-label
